description de l'intervention

L'API Gateway est une brique essentielle qui doit être extrêmement fiable et performante. Rust, grâce à son efficacité et sa fiabilité, est parfaitement adapté à cette mission. 
Nous verrons comment celà a été mis en place chez Apollo :

  • Les avantages et inconvénients d'une migration vers Rust.
  • La stratégie pour réécrire progressivement une grande base de code en Rust.
  • Les outils pour garantir que le nouveau code reste conforme au comportement de l'ancien.
  • Les techniques de débogage, de profilage et de surveillance d'une base de code en Rust.
